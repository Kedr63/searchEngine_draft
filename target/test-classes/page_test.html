Методические мероприятие по работе с генеративными нейронными сетями


Семинар «Удивительный нейромир. Работа с генеративными нейронными сетями»

Федоров Андрей Олегович, кандидат педагогических наук, доцент, главный аналитик отдела информационных систем и электронных ресурсов библиотеки НИУ "Высшая школа экономики"

Искусственный интеллект действительно открыл новые горизонты для библиотек, позволяя оптимизировать процессы и улучшать качество обслуживания пользователей. На семинаре мы обсудили, как использовать возможности ИИ для развития библиотек, а также определили задачи, которые можно доверить ИИ, и те, где незаменим человек. Найти баланс между технологиями и традициями стало ключевой задачей, требующей внимательного подхода.

На семинаре мы рассмотрели следующие направления:
  1. Составление промптов для библиотечной работы
  2. Запрос на составление аннотации к книге или статье.
  3. Генерация стандартных писем и уведомлений для читателей.
  4. Создание сценариев мероприятий, афиш, постов для соцсетей.
  5. Генеративные нейросети в библиотечной практике
  6. Автоматизация рутинных процессов: индексация, классификация, создание метаданных, поиск литературы.
  7. Креативные проекты: генерация иллюстраций для выставок, разработка уникального контента для продвижения библиотеки.
С методическими рекомендациями можно ознакомиться по ссылке.
